2001 yılında Denizli' de doğdum. İlk ve ortaöğretim kademelerini Denizli' de tamamladıktan sonra 2021 yılında Süleyman Demirel Üniversitesi Bilgisayar Mühendisliği bölümüne yerleştim. Şu an 3. sınıftayım ve öğrenimime devam etmekteyim. Java dilinde Spring frameworkü üzerinde çalışmaktayım ve ayrıca mobil uygulama geliştirme kısmında da Java kullamaktayım. Bununla beraber Android için Google' ın önermiş olduğu Kotlin dili üzerinde de çalışıyorum. Mobil uygulama geliştirme, grafik tasarım, GUI ve uygulamalarımın beckend kısmını Java dili ile programlamaktayım. Web alanına ise yeni giriş yaptım. HTML, CSS ve JavaScript üzerinde çalışarak kendimi bu konuda da ileriye götürmeye çalışıyorum.
